# Agent Birth

This file is first-run bootstrap only.

## Phase 1. Environment

1. Read the runtime router.
2. Read `agent-ir.json`.
3. Detect the active runtime before project inspection.
4. Ask before merging the common router into a native instruction file.
5. Ask before adapting skills, commands, or checkers into native mechanisms.
6. Verify required runtime files and optional `FILES.sha256`.
7. Remove import-only files only after verification and according to package policy.

Stop if environment decisions are missing.

## Phase 2. Project

1. Ask only for missing project blockers.
2. Inspect approved project sources.
3. Create or update project context.
4. Prepare workspace folders.
5. Save local technical facts to local memory.
6. Stop after birth. Wait for the user's next explicit task.

Do not make adapter preservation, connector preservation, or multi-platform export maintenance the installed agent's mission.
