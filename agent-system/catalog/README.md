# Repo/Tool Catalog

This folder stores external repositories and tools as a searchable idea library for future agent-building work.

Cataloging is not adoption. Entries here are not dependencies, runtime tools, prompts, MCP servers, or workflow integrations until a specific project selects them and passes license, security, runtime, validation, and rollback checks.

Files:

- `repo-tool-library.json` - machine-readable catalog.
- `repo-tool-cards/` - human-readable cards for individual entries.
- `repo-tool-intake-2026-07-07.md` - intake notes for the first batch of links.
